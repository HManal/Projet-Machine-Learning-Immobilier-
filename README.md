# Projet-Machine-Learning-Immobilier-
Développement d’un modèle de machine learning visant à prédire le prix de l’immobilier au Maroc
en fonction des caractéristiques du logement (surface, localisation, nombre de pièces, etc.).

Résultats obtenus :

- Conception d’un modèle de prédiction des prix immobiliers basé sur les caractéristiques du logement.

- Mise en évidence de l’influence de variables clés : surface, localisation, nombre de pièces, équipements.

- Obtention de prédictions cohérentes avec les tendances du marché immobilier marocain.

- Amélioration de la transparence et de l’aide à la décision pour acheteurs et investisseurs.

- Illustration concrète de l’apport du machine learning dans l’analyse des marchés immobiliers.
